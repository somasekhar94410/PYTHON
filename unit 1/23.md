# Tutorial Task 23: Income Tax Calculator

## 1. Problem Statement

Write a Python program to calculate the income tax payable by a person based on their annual income and tax percentage.

---

## 2. Algorithm

1. Start the program.
2. Read the annual income from the user.
3. Read the tax rate percentage.
4. Calculate the income tax amount.
5. Calculate the remaining income after tax deduction.
6. Display the tax amount and net income.
7. Stop the program.

---

---

## 4. Python Source Code

```python
annual_income = float(input("Enter Annual Income: "))
tax_rate = float(input("Enter Tax Rate (%): "))

tax_amount = (annual_income * tax_rate) / 100
net_income = annual_income - tax_amount

print("Income Tax =", tax_amount)
print("Net Income After Tax =", net_income)
```

---

## 5. Sample Input / Output

### Input

```text
Enter Annual Income: 600000
Enter Tax Rate (%): 10
```

### Output

```text
Income Tax = 60000.0
Net Income After Tax = 540000.0
```
