# Tutorial Task 26: GST Billing System

## 1. Problem Statement

Write a Python program to calculate the final bill amount by adding GST (Goods and Services Tax) to the original purchase amount.

---

## 2. Algorithm

1. Start the program.
2. Read the original bill amount.
3. Read the GST percentage.
4. Calculate the GST amount.
5. Calculate the final bill amount.
6. Display the GST amount and final bill amount.
7. Stop the program.

---

## 4. Python Source Code

```python
bill_amount = float(input("Enter Original Bill Amount: "))
gst_percentage = float(input("Enter GST Percentage: "))

gst_amount = (bill_amount * gst_percentage) / 100
final_bill = bill_amount + gst_amount

print("GST Amount =", gst_amount)
print("Final Bill Amount =", final_bill)
```
